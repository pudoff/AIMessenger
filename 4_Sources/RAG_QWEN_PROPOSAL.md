# RAG для AI Ассистента через QWEN API

## Идея

AI Ассистент может отвечать не только по одной фразе пользователя, а по контексту из базы: релевантные сообщения, чаты, участники, классификации и даты. Backend уже хранит embeddings сообщений в `MessageEmbedding` и умеет искать через pgvector, поэтому RAG можно строить поверх существующего `/api/search/semantic/`.

## Поток

1. Пользователь в разделе AI Ассистент выбирает "Запрос в свободной форме".
2. Frontend отправляет запрос на новый backend endpoint, например `POST /api/assistant/rag/`.
3. Backend строит embedding запроса через `ml_service.embed_text`.
4. Backend выбирает топ-N сообщений из доступных пользователю чатов через pgvector.
5. Backend собирает краткий контекст: текст сообщения, чат, автор, дата, классификация, similarity score.
6. Backend отправляет в QWEN API системный prompt, вопрос пользователя и найденный контекст.
7. Ответ QWEN возвращается пользователю вместе со списком использованных источников.

## Минимальный API

```http
POST /api/assistant/rag/
Content-Type: application/json

{
  "query": "Какие задачи по демо еще не закрыты?",
  "limit": 12,
  "chat": 10
}
```

Ответ:

```json
{
  "answer": "Краткий отчет ассистента...",
  "sources": [
    {
      "message_id": 50,
      "chat_id": 10,
      "chat_title": "Sprint planning",
      "text": "Нужно подготовить демо",
      "similarity_score": 0.83
    }
  ]
}
```

## Что важно учесть

- Фильтровать источники теми же правами доступа, что и `/api/search/semantic/`.
- Не отправлять в QWEN больше контекста, чем нужно: топ 8-15 сообщений обычно достаточно.
- Логировать ID источников и latency, но не писать полный prompt с приватными сообщениями в обычные логи.
- Добавить таймаут и graceful fallback: если QWEN недоступен, вернуть найденные источники и понятное сообщение.
- Вынести ключ QWEN в env: `QWEN_API_KEY`, `QWEN_BASE_URL`, `QWEN_MODEL`.

## UI

В AI Ассистенте можно оставить две кнопки:

- "Поиск по чатам" - быстрый semantic search без генерации ответа.
- "Запрос в свободной форме" - RAG-ответ через QWEN по найденному контексту.
