# Frontend - AIMessenger

Фронтенд-приложение на React для проекта AIMessenger — чата на базе искусственного интеллекта.

## Стек технологий

- **Фреймворк**: React 18.3.1
- **Сборщик**: Vite 5.4.10
- **Маршрутизация**: React Router DOM 6.30.1
- **Язык**: JavaScript (ES Module)

## Предварительные требования

- Node.js (v16 или выше)
- Менеджер пакетов npm или pnpm

## Быстрый старт

### Установка

```bash
cd 4_Sources/frontend
npm install
```

Или с использованием pnpm:

```bash
pnpm install
```

### Сервер разработки

Запуск сервера разработки с горячей перезагрузкой:

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173` (порт Vite по умолчанию).

### Сборка для продакшена

```bash
npm run build
```

Результат будет сгенерирован в директории `dist/`.

### Предварительный просмотр продакшен-сборки

```bash
npm run preview
```

## Структура проекта

```text
src/
├── api/                    # Утилиты API-клиента
├── components/             # Переиспользуемые React-компоненты
├── context/                # React Context для управления состоянием
├── data/                   # Тестовые данные и константы
├── layouts/                # Компоненты макетов (AppLayout и др.)
├── pages/                  # Компоненты страниц
├── styles/                 # Глобальные стили и CSS
├── utils/                  # Вспомогательные функции
├── assets/                 # Статические ресурсы
├── App.jsx                 # Главный компонент App
├── main.jsx                # Точка входа в приложение
├── .env                    # Переменные окружения
└── vite.config.js          # Конфигурация Vite
```

## Конфигурация

### Переменные окружения

Создайте файл `.env` в директории `src/` (или скопируйте из `.env.example`):

```env
VITE_API_BASE_URL=/api
```

Доступные переменные:
- `VITE_API_BASE_URL`: Базовый URL для конечных точек API

## Поддержка Docker

Приложение может быть контейнеризировано и развернуто с помощью Docker.

### Сборка Docker-образа

```bash
docker build -t slon-ai-messenger:latest .
```

### Запуск контейнера

```bash
docker run -p 80:80 slon-ai-messenger:latest
```

Подробная конфигурация контейнера находится в файле `Dockerfile`.

### Docker Compose

Используйте `docker-compose.yml` из корня проекта для развертывания полного стека:

```bash
docker-compose up frontend
```

## Зависимости

### Продакшен
- `react@^18.3.1` — библиотека пользовательского интерфейса
- `react-dom@^18.3.1` — рендеринг React DOM
- `react-router-dom@^6.30.1` — клиентская маршрутизация

### Разработка
- `vite@^5.4.10` — инструмент сборки и сервер разработки
- `@vitejs/plugin-react@^4.4.1` — поддержка React для Vite

## Рабочий процесс

1. **Разработка**: используйте `npm run dev` для локальной разработки с горячей перезагрузкой.
2. **Тестирование**: запуск тестов (если настроены).
3. **Сборка**: создание готового к продакшену бандла с помощью `npm run build`.
4. **Развертывание**: развертывание папки `dist/` на вашей хостинг-платформе.

## Примечания

- Горячая замена модулей (HMR) включена для удобной разработки.
- Приложение использует оптимизированный конвейер сборки Vite для быстрой упаковки.
- Запросы к API проксируются через настроенный базовый URL.
- Интеграция с бэкендом и другими сервисами описана в корневом `docker-compose.yml`.
