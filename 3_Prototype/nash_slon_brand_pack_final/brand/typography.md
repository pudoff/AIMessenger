# Типографика

## Заголовок / логотип
- Стиль: псевдо-винтажный serif.
- Рекомендация для сайта: `Playfair Display Bold`.
- Особенность логотипа: буква «О» в слове «слон» содержит хобот, вписанный внутрь контура буквы.

## Подпись
- `мессенджер`: `Inter Medium`, разрядка букв.
- Слоган: `Inter SemiBold`, красный акцент.

Google Fonts:
```html
<link href="https://fonts.googleapis.com/css2?family=Playfair+Display:wght@700&family=Inter:wght@400;500;600&display=swap" rel="stylesheet">
```
